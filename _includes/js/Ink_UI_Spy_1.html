Method name Description
Spy is a component that 'spies' an element (or a group of elements) and when they leave the viewport (through the top), highlight an option - related to that element being spied - that resides in a menu, initially identified as target.

Ink.UI.Spy (selector, [options], ) constructor 

  • String|DOMElement
  • Object
    Options 
  • DOMElement|String
    Target menu on where the spy will highlight the right option.

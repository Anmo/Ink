.map (array, map, [context], array, test, [context]) function

  • Arguments:
  • array
  • map
  • [ context ]
  • array
  • test
  • [ context ]

Runs a function for each item in the array. That function will receive each item as an argument and its return value will change the corresponding array item.

Accepts

Array
The array to map over
Function
The map function. Will take `(item, index, array)` as arguments and `this` will be the `context` argument.
Object
Object to be `this` in the map function.
Array
The array to filter
Function
A test function taking `(item, index, array)`
Object
Object to be `this` in the test function.

Returns

Array

Example

{% highlight html %}{% raw %} InkArray.map([1, 2, 3, 4], function (item) { return item + 1; }); // -> [2, 3, 4, 5] {% endraw %}{% endhighlight %} {% highlight html %}{% raw %} InkArray.filter([1, 2, 3, 4, 5], function (val) { return val > 2; }) // -> [3, 4, 5] {% endraw %}{% endhighlight %}

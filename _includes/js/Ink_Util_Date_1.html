Ink.Util.Date.get (format, [_date]) method 

  • String
    - as the string in which the date it will be formatted - mandatory 
  • Date
    - the date to format. If undefined it will do it on now() date. Can receive unix timestamp or a date object
Returns 
String

Function that works exactly as php date() function Works like PHP 5.2.2 PHP Date function

Examples:


			
    <script>
        Ink.requireModules( ['Ink.Util.Date_1'], function( InkDate ){
            console.log( InkDate.get('Y-m-d') ); // Result (at the time of writing): 2013-05-07
        });
    </script>

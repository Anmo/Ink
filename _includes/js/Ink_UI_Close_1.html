Ink.UI.Close_1

Subscribes clicks on the document.body. If and only if you clicked on an element having class "ink-close" or "ink-dismiss", will go up the DOM hierarchy looking for an element with any of the following classes: "ink-alert", "ink-alert-block". If it is found, it is removed from the DOM. One should call close once per page (full page refresh).

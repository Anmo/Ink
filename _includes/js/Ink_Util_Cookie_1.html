Ink.Util.Cookie.get (name) method

Accepts

String

Returns

String|Object

Example

Accepts

String

String

Ink.Util.Cookie.set (name, value, [expires], [path], [domain], [secure]) method

Sets a cookie

Accepts

String
Cookie name
String
Cookie value
Number
Number to add to current Date in seconds
String
Path to sets cookie (default '/')
String
Domain to sets cookie (default current hostname)
Boolean
True if wants secure, default 'false'

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Util.Cookie_1'], function( InkCookie ){ var expireDate = new Date( 2014,00,01, 0,0,0); InkCookie.set( 'someVarThere', 'anyValueHere', expireDate.getTime() ); }); {% endraw %}{% endhighlight %}

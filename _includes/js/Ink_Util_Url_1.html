.getAnchor ([str]) method

  • Arguments:
  • [ str ]

Get URL hash

Accepts

String
URL String. If not set, it will get the current URL.

Returns

String|Boolean

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Util.Url_1'], function( InkUrl ){ var anchor = InkUrl.getAnchor( 'http://www.sapo.pt/page.php#TEST' ); console.log( anchor ); // Result: TEST }); {% endraw %}{% endhighlight %}

.getAnchorString ([string]) method

  • Arguments:
  • [ string ]

Get anchor string of current or passed URL

Accepts

String
If not provided it uses the current URL.

Returns

Object

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Util.Url_1'], function( InkUrl ){ var hashParams = InkUrl.getAnchorString( 'http://www.sapo.pt/#var1=valueVar1&var2=valueVar2' ); console.log( hashParams ); // Result: // { // var1: 'valueVar1', // var2: 'valueVar2' // } }); {% endraw %}{% endhighlight %}

.parseUrl (url) method

  • Arguments:
  • url

Parse passed URL

Accepts

String
URL to be parsed

Returns

Object

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Util.Url_1'], function( InkUrl ){ var parsedURL = InkUrl.parseUrl( 'http://www.sapo.pt/index.html?var1=value1#anchor' ) console.log( parsedURL ); // Result: // { // 'scheme' => 'http', // 'host' => 'www.sapo.pt', // 'path' => '/index.html', // 'query' => 'var1=value1', // 'fragment' => 'anchor' // } }); {% endraw %}{% endhighlight %}

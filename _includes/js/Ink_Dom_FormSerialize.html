Ink.Dom.FormSerialize

METHODS
Method name Description
.fillIn Sets form elements's values with values given from object One cannot restore the values of an input with `type="file"` (browser prohibits it)
.serialize Serializes a form into an object, turning field names into keys, and field values into values. note: Multi-select and checkboxes with multiple values will yield arrays
Supports serialization of form data to/from javascript Objects. Valid applications are ad hoc AJAX/syndicated submission of forms, restoring form values from server side state, etc.

Ink.UI.TreeView class

Shows elements in a tree-like hierarchical structure.

Defined methods:

  • Ink.UI.TreeView (selector, )

Ink.UI.TreeView (selector, ) constructor 

  • String|DOMElement
  • String
    Selector to define which elements are seen as nodes. 
  • String
    Selector to define which elements are represented as children. 
  • String
    Classes to be added to the parent node. 
  • String
    icon-minus-circle'] Classes to be added to the icon when a parent is open. 
  • String
    icon-plus-circle'] Classes to be added to the icon when a parent is closed. 
  • String
    Class to toggle visibility of the children. 
  • String
    The name of icon tag. The component tries to find a tag with that name as a direct child of the node. If it doesn't find it, it creates it. 
  • Boolean
    Stops the default behavior of the click handler.

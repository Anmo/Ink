.add (element, [options], ) method

  • Arguments:
  • element
  • [ options ]

Makes an element droppable and adds it to the stack of droppable elements. Can consider it a constructor of droppable elements, but where no Droppable object is returned. In the following arguments, any events/callbacks you may pass, can be either functions or strings. If the 'move' or 'copy' strings are passed, the draggable gets moved into this droppable. If 'revert' is passed, an acceptable droppable is moved back to the element it came from.

Accepts

String|DOMElement
Target element
Object
options object
String
teste Classname(s) applied when an acceptable draggable element is hovering the element
String
teste Selector for choosing draggables which can be dropped in this droppable.
Function
teste callback called when an acceptable draggable element is hovering the droppable. Gets the draggable and the droppable element as parameters.
Function|String
teste callback called when an acceptable draggable element is dropped. Gets the draggable, the droppable and the event as parameters.
Function|String
teste callback called when a droppable is dropped outside this droppable. Gets the draggable, the droppable and the event as parameters. (see above for string options).

Ink.UI.Droppable_1 version: 1

METHODS
Method name Description
.add Makes an element droppable and adds it to the stack of droppable elements. Can consider it a constructor of droppable elements, but where no Droppable object is returned. In the following arguments, any events/callbacks you may pass, can be either functions or strings. If the 'move' or 'copy' strings are passed, the draggable gets moved into this droppable. If 'revert' is passed, an acceptable droppable is moved back to the element it came from.
.remove Removes an element from the droppable stack and removes the droppable behavior

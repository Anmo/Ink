Ink.UI.Modal (selector, [options], ) constructor 

  • String|DOMElement
  • Object
    Options 
  • String
    Default/Initial width. Ex: '600px' 
  • String
    Default/Initial height. Ex: '400px' 
  • String
    Custom class to be added to the div.ink-shade 
  • String
    Custom class to be added to the div.ink-modal 
  • String
    CSS Selector to target elements that will trigger the Modal. 
  • String
    Trigger's event to be listened. 'click' is the default value. Ex: 'mouseover', 'touchstart'... 
  • Boolean
    Display the Modal automatically when constructed. 
  • String
    Markup to be placed in the Modal when created 
  • Function
    Callback function to run when the Modal is opened. 
  • Function
    Callback function to run when the Modal is closed. Return `false` to cancel dismissing the Modal. 
  • Boolean
    Determines if the Modal should close when clicked outside of it. 
  • Boolean
    Determines if the Modal should close when "Esc" key is pressed. 
  • Boolean
    Determines if the Modal should behave responsively (adapt to smaller viewports). 
  • Boolean
    Determines if the Modal should 'disable' the page's scroll (not the Modal's body).

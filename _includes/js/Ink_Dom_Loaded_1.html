Ink.Dom.Loaded class

1

The Loaded class provides a method that allows developers to queue functions to run when the page is loaded (document is ready).

Defined methods:

  • Ink.Dom.Loaded.run ([win], fn)

Ink.Dom.Loaded.run ([win], fn) method 

  • Object
    Window object to attach/add the event 
  • Function
    Callback function to be run after the page is loaded

Adds a new function that will be invoked once the document is ready

Examples:


			
    Ink.requireModules(['Ink.Dom.Loaded_1'], function(Loaded){
        Loaded.run(function(){
            console.log('This will run when the page/document is ready/loaded');
        });
    });

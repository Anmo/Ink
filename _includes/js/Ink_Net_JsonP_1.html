Ink.Net.JsonP class

This class takes care of the nitty-gritty details of doing jsonp requests: Storing a callback in a globally accessible manner, waiting for the timeout or completion of the request, and passing extra GET parameters to the server, is not so complicated but it's boring and repetitive to code and tricky to get right.

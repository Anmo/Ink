Ink.UI.Toggle (selector, [options], ) constructor 

  • String|DOMElement
  • Object
    Options 
  • String
    CSS Selector that specifies the elements that this component will toggle 
  • String
    className when toggle is On 
  • String
    className when toggle is Off. 
  • String
    Event that will trigger the toggling. 
  • Boolean
    When this is on, if the user clicks outside of the toggled content, the target is toggled off. 
  • Selector
    Toggle off when an element matching this selector is clicked. Set to null to deactivate the check. Default: 'a[href]' (finds links) 
  • Boolean
    Whether to start toggled off, on, or as found in the markup. (false: off, true: on, null: markup) 
  • Function
    Callback to be called when the toggle state changes. Return `false` to cancel the event.

Introduction

You'll find navigation blocks very easy and intuitive to implement. Like any other Ink component, its construction is a combination of simple and semantic markup with natural language class naming.

Our default types of navigation are:

  • menu for default navigation blocks.
  • breadcrumbs for breacrumb navigation.
  • pagination for pagination of tables, carousels, galeries, etc...
  • pills for pill blocks.

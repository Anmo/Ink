Droppables

You can have elements which can be dragged around by your user.

Example

Drop 1 here

Drop 2 here

Drag me (1) Drag me (2)

Code

  1. <p>
  2.     Drop 1 <b id="drop">here</b>
  3. </p>
  4. <p>
  5.     Drop 2 <b id="drop2">here</b>
  6. </p>
  7. <span id="dropex-1" class="acceptme">
  8.     Drag me (1)
  9. </span>
  10. <span id="dropex-2" class="acceptme2">
  11.     Drag me (2)
  12. </span>
  13. <script type="text/javascript">
  14.     Ink.requireModules(['Ink.UI.Draggable_1', 'Ink.UI.Droppable_1'], function (Draggable, Droppable) {
  15.         new Draggable('#dropex-1', {});
  16.         new Draggable('#dropex-2', {});
  17.         Droppable.add('#drop', {
  18.             hoverclass: 'ink-label success',
  19.             accept: '.acceptme',  // Select acceptable elements here
  20.         });
  21.         Droppable.add('#drop2', {
  22.             hoverclass: 'ink-label success',
  23.             accept: '.acceptme2',
  24.             onDrop: 'move',  // This is a predefined event handler which moves the draggable element here when dropped
  25.             onDropOut: 'revert'  // This predefined event handler moves the draggable back to where it came from
  26.         });
  27.     });
  28. </script>

Options

Droppable options:

  • element target element
  • options options object
  • hoverClass Classname applied when an acceptable draggable element is hovering the element
  • accept Selector for elements that can be accepted by this droppable
  • Event handlers: Any of the following can take a function, or one of the strings 'revert' (to place the draggable where it came from, useful in onDropOut) 'move' (to move the element) 'copy' (to copy the element)
    • onHover callback called when an acceptable draggable element is hovering the droppable. Gets the draggable and the droppable element as parameters.
    • onDrop callback called when an acceptable draggable element is dropped. Gets the draggable, the droppable and the event as parameters.
    • onDropOut callback called when an acceptable draggable element is dropped. Gets the draggable, the droppable and the event as parameters.

More details can be found in the technical documentation.

More than HTML and CSS

We've developed our own Javascript library based on the latest coding standards and a modular architecture. Using Ink.requireModules() method make sure all of Ink power is at your finger tips.

{% highlight js %} Ink.requireModules( ['Ink.Dom.Css_1'], function(css) { var elm = Ink.s('.ink'); css.addClassName(elm,'hot'); } ); {% endhighlight %}

Customizable

Inks CSS is created with SASS. Breakpoints, dimensions, color, font sizes and even class names are defined as variables and are easily reconfigurable. Just edit a couple of files, recompile and you'll have your unique build.

{% highlight scss %} // scss $ink-class-prefix: my-; $navigation-class-name: #{$ink-class-prefix}nav // css .my-nav { ... } {% endhighlight %}

How?

Generated

Inks CSS is created with SASS and uses Compass to make our work a lot easier. This also makes it very easy for you to do the same.

Customizable

Allmost all aspects of Ink, like dimensions, coloring, typography and even class names are defined with SASS variables. Creating your own version is as simple as editing a few files and recompiling.

Modular

If you need just a couple of our components, just edit the code and recompile your stripped down version of our CSS.

Javascript

Ink.js is our own library and comes packed with lots of UI components, and the tools to let you build your own.

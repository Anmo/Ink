Accepts

String|DOMElement
String CSS class added to the "ghost" element being dragged around. Defaults to 'placeholder'.
String CSS class added to the original element being dragged around. Defaults to 'hide-all'.
String CSS class added to the html element when the user is dragging. Defaults to 'dragging'.
String CSS selector for the drag enabled nodes. Defaults to 'li'.
String CSS selector for the drag handle. If present, you can only drag nodes by this selector.
String CSS selector to validate a node move. If present, you can only move nodes inside this selector.
Boolean Flag to swap dragged element and target element instead of reordering it.
Boolean Flag to cancel draggin if mouse leaves the container element.

Methods
Method name Description
(constructor) Constructor

Ink.UI.ImageQuery(selector, [options], ) constructor

  • Arguments:
  • selector
  • [options]

Accepts

  • selector String|DOMElement

    Selector or element

  • [options] Object

    Options object

  • [options.src] String|Function

    String or Callback function (that returns a string) with the path to be used to get the images.

  • [options.retina] String|Function

    String or Callback function (that returns a string) with the path to be used to get RETINA specific images.

  • [options.queries] Array

    Array of queries

  • [options.queries.label] String

    Label of the query. Ex. 'small'

  • [options.queries.width] Number

    Min-width to use this query

  • [options.onLoad] Function

    Date format string

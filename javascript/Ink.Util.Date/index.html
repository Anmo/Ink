.get(format, [_date]) method

  • Arguments:
  • format
  • [_date]

Formats a date object.
This works exactly as php date() function. http://php.net/manual/en/function.date.php

Accepts

  • format String

    The format in which the date it will be formatted.

  • [_date] Date

    The date to format. Can receive unix timestamp or a date object. Defaults to current time.

Returns

  • String

    Formatted date

  • Arguments:

Creates a date object based on a format string.

Accepts

  • [format] String

    The format in which the date will be formatted. Defaults to 'Y-m-d'
  • str_date String

    The date formatted.

Returns

  • Date

    Date object based on the formatted date and format

All samples are using Ink.requireModules, please read how to use it at Ink.requireModules section

Functions
Function name Description
.get Gets an object with the current page cookies.
.remove Deletes a cookie.
.set Sets a cookie.

.get(name) method

  • Arguments:
  • name

Gets an object with the current page cookies.

Accepts

  • name String

    The cookie name.

Returns

  • String|Object

    If the name is specified, it returns the value of that key. Otherwise it returns the full cookie object

.remove(cookieName, [path], [domain]) method

  • Arguments:
  • cookieName
  • [path]
  • [domain]

Deletes a cookie.

Accepts

  • cookieName String

    Cookie name.

  • [path] String

    Path of the cookie. Defaults to '/'.

  • [domain] String

    Domain of the cookie. Defaults to current hostname.

.set(name, value, [expires], [path], [domain], [secure]) method

  • Arguments:
  • name
  • value
  • [expires]
  • [path]
  • [domain]
  • [secure]

Sets a cookie.

Accepts

  • name String

    Cookie name.
  • value String

    Cookie value.

  • [expires] Number

    Number of seconds the cookie will be valid for.

  • [path] String

    Path for the cookie. Defaults to '/'.

  • [domain] String

    Domain for the cookie. Defaults to current hostname.

  • [secure] Boolean

    Flag for secure. Default 'false'.

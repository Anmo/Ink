.get(name)function

  • Arguments:
  • name

Gets an object with the current page cookies.

Accepts

String
The cookie name.

Returns

String|Object

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Util.Cookie_1'], function( InkCookie ){ var myCookieValue = InkCookie.get('someVarThere'); console.log( myCookieValue ); // This will output the value of the cookie 'someVarThere', from the cookie object. }); {% endraw %}{% endhighlight %}

  • name

  • [path]

Sets a cookie.

Accepts

String

String

Boolean

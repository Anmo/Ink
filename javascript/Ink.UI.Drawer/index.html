Ink.UI.Drawer_1

Displays off-canvas content which can be triggered by clicking elements with the 'left-drawer-trigger' and 'right-drawer-trigger', respectively.
The left drawer has the 'left-drawer' class, and the right drawer has the 'right-drawer' class. The content drawer (EG your <div id="main">) must have the 'content-drawer' class. For more, see the example below, or try the sample.

All samples are using Ink.requireModules, please read how to use it at Ink.requireModules section

Accepts

  • [options] Object

    Configuration options.

  • [options.closeOnContentClick] Boolean

    Flag to close the drawer when someone clicks on the `.contentDrawer`

  • [options.mode] String

    This can be 'push' or 'hide'.

  • [options.sides] String

    Can be 'left', 'right', or 'both'. Controls what sides have a drawer.

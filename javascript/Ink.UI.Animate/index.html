Ink.UI.Animate_1(element, options, ) constructor

  • Arguments:
  • element
  • options

Accepts

  • element DOMElement

    Animated element
  • options Object

    Options object
  • options.animation String

    Animation name

  • [options.duration] String|Number

    Duration name (fast|medium|slow) or duration in milliseconds. Defaults to 'medium'.

  • [options.removeClass] Boolean

    Flag to remove the CSS class when finished animating. Defaults to false.

  • [options.onEnd] Function

    Callback for the animation end

.animate(element, animation, [options], ) method

  • Arguments:
  • element
  • animation
  • [options]

Accepts

  • element

    {DOMElement} animated element
  • animation

    {String} animation name

  • [options]

    {Object}

  • [options.onEnd]

    null {Function} callback for animation end

  • [options.removeClass]

    false {Boolean} whether to remove the Css class when finished

  • [options.duration]

    medium {String|Number} duration name (fast|medium|slow) or duration in ms

Example

{% highlight html %}{% raw %} Animate.animate(myDiv, 'fadeIn') {% endraw %}{% endhighlight %} {% highlight html %}{% raw %} Animate.animate(myDiv, 'shake', { onEnd: function () { alert('Finished shaking!'); } }); {% endraw %}{% endhighlight %}

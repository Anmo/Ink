Ink.UI.Toggle(selector, [options], ) constructor

  • Arguments:
  • selector
  • [options]

Accepts

String|DOMElementTrigger element. By clicking this, the target (or targets) are triggered.
ObjectOptions object, containing:
StringCSS Selector that specifies the elements that this component will toggle
StringCSS class to toggle when on. Defaults to 'show-all'.
StringCSS class to toggle when off. Defaults to 'hide-all'.
StringEvent that will trigger the toggling. Defaults to 'click'.
BooleanFlag to toggle the targe off when clicking outside the toggled content. Defaults to true.
StringToggle off when an element matching this selector is clicked. Set to null to deactivate the check. Defaults to 'a[href]'.
BooleanFlag to define initial state. false: off, true: on, null: markup. Defaults to null.
FunctionCallback when the toggle state changes. Return `false` to cancel the event.

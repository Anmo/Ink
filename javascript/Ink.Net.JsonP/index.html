Ink.Net.JsonP_1

Executes a JSONP request

All samples are using Ink.requireModules, please read how to use it at Ink.requireModules section

Methods
Method name Description
(constructor) Constructor

Ink.Net.JsonP(uri, options, ) constructor

  • Arguments:
  • uri
  • options

Accepts

  • uri String

    Request URL
  • options Object

    Request options
  • options.onSuccess Function

    Success callback

  • [options.onFailure] Function

    Failure callback

  • [options.failureObj] Object

    Object to be passed as argument to failure callback

  • [options.timeout] Number

    Timeout for request fail, in seconds. defaults to 10

  • [options.params] Object

    Object with the parameters and respective values to unfold

  • [options.callbackParam] String

    Parameter to use as callback. defaults to 'jsoncallback'

  • [options.internalCallback] String

    Name of the callback function stored in the Ink.Net.JsonP object.

Ink.UI.Carousel_1(selector, [options], ) constructor

  • Arguments:
  • selector
  • [options]

Accepts

String|DOMElementDOM element or element id
ObjectCarousel Options
IntegerMilliseconds to wait before auto-advancing pages. Set to 0 to disable auto-advance. Defaults to 0.
StringAxis of the carousel. Set to 'y' for a vertical carousel. Defaults to 'x'.
BooleanFlag to center the carousel horizontally.
NumberInitial index page of the carousel. Defaults to 0.
BooleanIf there are not enough slides to fill the full width of the last page, leave white space. Defaults to `true`.
BooleanEnable swipe support if available. Defaults to true.
MixedEither an ul element to add pagination markup to or an `Ink.UI.Pagination` instance to use.
FunctionCallback to be called when the page changes.

.refit() method

Repositions elements around. Measure the carousel once again, adjusting the involved elements' sizes. This is called automatically when the window resizes, in order to cater for changes from responsive media queries, for instance.

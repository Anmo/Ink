.changeFontSize(selector, delta, [op], [minVal], [maxVal]) method

  • Arguments:
  • selector
  • delta
  • [op]
  • [minVal]
  • [maxVal]

Change the font size of elements. Changes the font size of the elements which match the given CSS rule For this function to work, the CSS file must be in the same domain than the host page, otherwise JS can't access it.

Accepts

StringCSS selector rule
NumberNumber of pixels to change on font-size
StringSupported operations are '+' and '*'. defaults to '+'
NumberIf result gets smaller than minVal, change does not occurr
NumberIf result gets bigger than maxVal, change does not occurr

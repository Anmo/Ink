.changeFontSize(selector, delta, [op], [minVal], [maxVal]) method

  • Arguments:
  • selector
  • delta
  • [op]
  • [minVal]
  • [maxVal]

Change the font size of elements.
Changes the font size of the elements which match the given CSS rule
For this function to work, the CSS file must be in the same domain than the host page, otherwise JS can't access it.

Accepts

String CSS selector rule
Number Number of pixels to change on font-size
String Supported operations are '+' and '*'. defaults to '+'
Number If result gets smaller than minVal, change does not occurr
Number If result gets bigger than maxVal, change does not occurr

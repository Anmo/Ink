Ink.UI.Tooltip(target, [options], ) constructor

  • Arguments:
  • target
  • [options]

Accepts

DOMElement|StringTarget element or selector of elements, to display the tooltips on.
ObjectOptions object
StringText content for the tooltip.
StringHTML for the tooltip. Same as above, but won't escape HTML.
StringPositioning for the tooltip. Options are 'up', 'down', 'left', 'right', 'mousemove' (follows the cursor), and 'mousefix' (stays fixed). Defaults to 'up'.
StringColor of the tooltip. Options are red, orange, blue, green and black. Default is white.
NumberNumber of seconds to fade in/out. Defaults to 0.3.
BooleanFlag to prevent the tooltip from being erased when the mouse hovers away from the target.
NumberNumber of seconds the tooltip will stay open. Useful together with options.forever. Defaults to 0.
NumberTime the tooltip waits until it is displayed. Useful to avoid getting the attention of the user unnecessarily
DOMElement|SelectorElement or selector containing HTML to be cloned into the tooltips. Can be a hidden element, because CSS `display` is set to `block`.
StringSelector within the template element to choose where the text is inserted into the tooltip. Useful when a wrapper DIV is required.
NumberSpacing from the target to the tooltip, when `where` is `mousemove` or `mousefix`. Defaults to 10.
NumberSpacing from the target to the tooltip, when `where` is `mousemove` or `mousefix`. Defaults to 10.
NumberSpacing between the tooltip and the target element, when `where` is not `mousemove` or `mousefix`. Defaults to 8.

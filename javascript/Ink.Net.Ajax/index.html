Ink.Net.Ajax(url, options, ) constructor

  • Arguments:
  • url
  • options

Accepts

StringRequest URL
ObjectRequest options
BooleanIf false, the request synchronous.
BooleanFlag to activate CORS. Set this to true if you're doing a cross-origin request
StringHTTP request method. POST by default.
Object|StringRequest parameters to be sent with the request
NumberRequest timeout in seconds
NumberArtificial delay. If the request is completed faster than this delay, wait the remaining time before executing the callbacks
StringPOST request body. If not specified, it's filled with the contents from parameters
StringContent-type header to be sent. Defaults to 'application/x-www-form-urlencoded'
ObjectKey-value pairs for additional request headers
FunctionCallback executed after the request is completed, regardless of what happened during the request.
FunctionCallback executed if the request is successful (requests with 2xx status codes)
FunctionCallback executed if the request fails (requests with status codes different from 2xx)
FunctionCallback executed if an exception occurs. Receives the exception as a parameter.
FunctionCallback executed after object initialization but before the request is made
FunctionCallback executed before any initialization
FunctionCallback executed if the request times out
Boolean|StringIf the request Content-type header is application/json, evaluates the response and populates responseJSON. Use 'force' if you want to force the response evaluation, no matter what Content-type it's using.
BooleanFlag to sanitize the content of responseText before evaluation
StringURI for proxy service hosted on the same server as the web app, that can fetch documents from other domains. The service must pipe all input and output untouched (some input sanitization is allowed, like clearing cookies). e.g., requesting http://example.org/doc can become /proxy/http%3A%2F%2Fexample.org%2Fdoc The proxy service will be used for cross-domain requests, if set, else a network error is returned as exception.

.run([win], fn)method

  • Arguments:

Specify a function to execute when the DOM is fully loaded.

Accepts

Object
Window object to attach/add the event
Function
Callback function to be executed after the DOM is ready

Example

{% highlight html %}{% raw %} Ink.requireModules(['Ink.Dom.Loaded_1'], function(Loaded){ Loaded.run(function(){ console.log('This will run when the page/document is ready/loaded'); }); }); {% endraw %}{% endhighlight %}

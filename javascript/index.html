Introduction

Inks JavaScript library extends native JavaScript functionality without interfering with native objects. It also provides a powerful abstraction layer that makes handling different implementations of the language very straightforward.

Structure

Our library is written in a modular fashion so that you can load only the necessary modules for a given task.

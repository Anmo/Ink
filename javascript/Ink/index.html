.bind(fn, context, [args*]) method

  • Arguments:
  • fn
  • context
  • [args*]

Function.prototype.bind alternative.
Creates a new function that, when called, has its this keyword set to the provided value, with a given sequence of arguments preceding any provided when the new function is called.

Accepts

Function The function
Object The value to be passed as the this parameter to the target function when the bound function is called. If used as false, it preserves the original context and just binds the arguments.
Any Additional arguments will be sent to the original function as prefix arguments.

Returns

Function

.createModule(mod, version, deps, modFn) method

  • Arguments:
  • mod
  • version
  • deps
  • modFn

Creates a new module.
Use this to wrap your code and benefit from the module loading used throughout the Ink library

Accepts

String Module name, separated by dots. Like Ink.Dom.Selector, Ink.UI.Modal
Number Version number
Array Array of module names which are dependencies of the module being created. The order in which they are passed here will define the order they will be passed to the callback function.
Function The callback function to be executed when all the dependencies are resolved. The dependencies are passed as arguments, in the same order they were declared. The function itself should return the module.

.extendObj(destination, source, [args*]) method

  • Arguments:
  • destination
  • source
  • [args*]

Extends an object with another
Copy all of the properties in one or more source objects over to the destination object, and return the destination object. It's in-order, so the last source will override properties of the same name in previous arguments.

Accepts

Object The object that will receive the new/updated properties
Object The object whose properties will be copied over to the destination object
Object Additional source objects. The last source will override properties of the same name in the previous defined sources

Returns

destination object, enriched with defaults from the sources

.setPath(key, rootURI) method

  • Arguments:
  • key
  • rootURI

Sets the URL path for a namespace.
Use this to customize where requireModules and createModule will load dependencies from.
This can be useful to set your own CDN for dynamic module loading or simply to change your module folder structure

Accepts

String Module or namespace
String Base URL path and schema to be appended to the module or namespace

Example

{% highlight html %}{% raw %} Ink.setPath('Ink', 'http://my-cdn/Ink/'); Ink.setPath('Lol', 'http://my-cdn/Lol/'); // Loads from http://my-cdn/Ink/Dom/Whatever/lib.js Ink.requireModules(['Ink.Dom.Whatever'], function () { ... }); // Loads from http://my-cdn/Lol/Whatever/lib.js Ink.requireModules(['Lol.Whatever'], function () { ... }); {% endraw %}{% endhighlight %}

.addItem(item) method

  • Arguments:
  • item

Adds an item to the end of your stacks. Call reloadItems() when you are done adding items.

Accepts

DOMElementElement

.reloadItems() method

Updates the layout of your items. Call this method after adding items or changing their dimensions. This method is automatically called when the window resizes.

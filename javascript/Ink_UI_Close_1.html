Ink.UI.Close_1 version: 1 Closing utilities

Subscribes clicks on the document.body. Whenever an ".ink-close,.ink-dismiss" element is clicked, this module finds an ancestor ".ink-alert,.ink-alert-block" element and removes it from the DOM. This module should be called only once per page.

Ink.UI.Pagination(selector, options, ) constructor

  • Arguments:
  • selector
  • options

Accepts

  • selector String|DOMElement

    Selector or element
  • options Object

    Options

  • [options.size] Number

    Number of pages.

  • [options.totalItemCount] Number

    Total numeber of items to display

  • [options.itemsPerPage] Number

    Number of items per page.

  • [options.maxSize] Number

    If passed, only shows at most maxSize items. displays also first|prev page and next page|last buttons

  • [options.start] Number

    Start page. defaults to 1

  • [options.sideButtons] Boolean

    true Whether to show the first, last, previous, next, previousPage and lastPage buttons. Do not use together with maxSize.

  • [options.firstLabel] String

    Text for the first page button. Defaults to 'First'.

  • [options.lastLabel] String

    Text for the last page button. Defaults to 'Last'.

  • [options.previousLabel] String

    Text for the previous button. Defaults to 'Previous'-

  • [options.nextLabel] String

    Text for the next button. Defaults to 'Next'

  • [options.previousPageLabel] String

    Text for the previous page button. Defaults to 'Previous {Items per page}'.

  • [options.nextPageLabel] String

    Text for the next page button. Defaults to 'Next {Items per page}'.

  • [options.onChange] Function

    Callback to be called when a page changes. Called with `(thisPaginator, newPageNumber)`.

  • [options.hashParameter] String

    Parameter to use on setHash. Defaults to 'page'.

  • [options.parentTag] String

    HTML Tag used as the parent node.

  • [options.childTag] String

    HTML Tag used as the child nodes.

  • [options.wrapperClass] String

    CSS Class used in the wrapper element

  • [options.paginationClass] String

    CSS Class used in the pagination element

  • [options.activeClass] String

    CSS Class used to mark page as active

  • [options.disabledClass] String

    CSS Class used to mark page as disabled

  • [options.hideClass] String

    CSS Class used to hide elements

  • [options.previousClass] String

    CSS Class used in the previous element

  • [options.previousPageClass] String

    CSS Class used in the previous page element

  • [options.nextClass] String

    CSS Class used in the next element

  • [options.nextPageClass] String

    CSS Class used in the next page element

  • [options.numberFormatter] Function

    Number formatter function. Receives a 0-indexed number and returns the text for the numbered page button.

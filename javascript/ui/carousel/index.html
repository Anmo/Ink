Carousel

The Carousel component is used to display more information than the screen might have space for, by paginating lists of DOM elements using intuitive pagination controls.

This control is deployed using markup and the ink-carousel class, so as to provide a useful display even when JavaScript is not available in the target browser.

Options

You can define options through the second argument of the constructor, or as data-attributes on the element.

Some options:

  • axis: Controls whether this is a vertical carousel or a horizontal carousel. Can by 'x' or 'y'.
  • pagination: An element which gets the pagination controls, or an Ink.UI.Pagination instance.

More details can be found in the technical documentation.

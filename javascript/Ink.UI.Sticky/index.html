Ink.UI.Sticky_1

Ink.UI.Sticky makes an element "stick" to the screen and stay in the same place as the user scrolls. To use it, just select an element as you create the Sticky. As you scroll past it, it will stick to the top of the screen.
The activateInLayouts option controls in what layouts this behaviour happens. By default, it is disabled for the small and tiny layouts. Pass a comma-separated string to choose just the layouts you need. You can use the offsetTop option if you want it to keep some distance from the top of the screen. To avoid it going under the footer of your page, pass a selector to your footer as the bottomElement option.

All samples are using Ink.requireModules, please read how to use it at Ink.requireModules section

(constructor) Constructor

Ink.UI.Sticky(selector, [options], ) constructor

  • Arguments:
  • selector
  • [options]

Accepts

  • selector String|DOMElement

    Element or selector

  • [options] Object

    Options Options object.

  • [options.offsetBottom] Number

    Number of pixels of distance from the bottomElement. Defaults to 0.

  • [options.offsetTop] Number

    Number of pixels of distance from the topElement. Defaults to 0.

  • [options.inlineDimensions] Boolean

    Set to false to disable setting inline CSS dimensions. Use this if you want to use CSS to define your own dimensions. Defaults to true.

  • [options.inlinePosition] Boolean

    Set to false to disable setting inline CSS positions. Use this if you want to use CSS to define your own positioning. Defaults to true.

  • [options.wrapperClass] String

    CSS class for the wrapper element. Defaults to 'ink-sticky-wrapper'.

  • [options.stickyClass] String

    CSS class to stick the element to the screen. Defaults to 'ink-sticky-stuck'.

  • [options.topElement] String

    CSS Selector that specifies a top element with which the component could collide.

  • [options.bottomElement] String

    CSS Selector that specifies a bottom element with which the component could collide.

  • [options.activateInLayouts] Array|String

    Layouts in which the sticky behaviour is present. Pass an array or comma-separated string. Defaults to 'tiny,small,medium,large,xlarge'.
